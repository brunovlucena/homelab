## Key Metrics

```promql
# Availability SLI
sli:availability:ratio = successful_requests / total_requests

# Latency SLI  
sli:latency:p95 = histogram_quantile(0.95, request_duration_seconds)

# Error budget remaining
error_budget:remaining = 1 - ((1 - sli:availability) / (1 - 0.995))
```
