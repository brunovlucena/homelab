# Doctor Mobile App

React Native mobile application for doctors to interact with their AI medical assistant.

## Features

- Real-time chat with agent-medical
- Case summarization
- Patient record access
- Medical data correlation
- Push notifications for alerts

## Setup

```bash
npm install
npx expo start
```

## Configuration

Set environment variables:
- `API_URL`: Backend API URL
- `WS_URL`: WebSocket URL for messaging
